추가 및 수정 내역
1. 국기 이미지 선명하게
국기 이미지가 흐리던 문제 해결.
이미지 크기를 키워서 더 선명하게 보이도록 변경.
2. 버튼과 시계 색상 맞춤
기존 코드에서 필터 버튼 색상(무지개색) 가져와 적용.
시계 숫자 색상을 필터 버튼 색상에 맞춤(초록, 노랑, 주황, 파랑, 남색, 보라).
시계 카드 배경색이 잘못 바뀌었던 문제 수정 → 원래 흰색으로 복원.
3. 필터 버튼 클릭 시 크기 변경
필터 버튼 클릭하면 살짝 커지게 설정.
현재 선택된 탭이 더 잘 보이도록 강조.
